# 8.1.1.13 EdgeChamfer

The `EdgeChamfer` specification defines board edge chamfering: angle, width, and which side(s) receive the chamfer.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                 | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|------------|
| EdgeChamfer              | EdgeChamferType          | Identifies an edge chamfer specification instance.                                                                                           | 0-n        |
| type                     | edgeChamferListType      | ANGLE | WIDTH | SIDE. Angle referenced from the plane of each identified layer; width is perpendicular from the edge; side identifies layer(s). | 1-1        |
| comment                  | string                   | Description supplementing/differing from the enumerations.                                                                                   | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes (e.g., angle, width, layer refs).                                                                                  | 0-n        |

Linking guidance: Use an Outline or Line on a layer with `layerFunction="EDGE_CHAMFER"`. Associate via SpecRef on the Layer or per LayerFeature:Set.

Example:
```xml
<Spec name="Edge_Connector_P1">
  <EdgeChamfer type="ANGLE">
    <Property value="20.0" unit="DEGREES" plusTol="5.0" minusTol="5.0" tolPercent="false"/>
  </EdgeChamfer>
  <EdgeChamfer type="WIDTH">
    <Property value="0.055" unit="INCH" plusTol="0.010" minusTol="0.010" tolPercent="false"/>
  </EdgeChamfer>
  <EdgeChamfer type="SIDE">
    <Property layerOrGroupRef="L1"/>
    <Property layerOrGroupRef="BOTTOM"/>
  </EdgeChamfer>
</Spec>
```
