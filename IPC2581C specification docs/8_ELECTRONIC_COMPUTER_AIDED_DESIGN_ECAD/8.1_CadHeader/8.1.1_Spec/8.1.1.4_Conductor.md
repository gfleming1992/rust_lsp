# 8.1.1.4 Conductor

The `Conductor` specification defines conductor characteristics such as material, surface conditions, etch factor, finished height, and roughness.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                              | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------|------------|
| Conductor                | ConductorType            | Identifies a conductor specification instance.                                                                            | 0-n        |
| material                 | conductorMaterialType    | COPPER | SILVER_INK | CARBON_INK | CONDUCTIVE_INK | OTHER.                                                                             | 0-1        |
| foilType                 | foilTypeType             | Copper foil type per IPC-4652 (e.g., CU-E1, CU-E2, ...).                                                                  | 0-1        |
| type                     | conductorListType        | CONDUCTIVITY | PRODUCT_NAME | SURFACE_ROUGHNESS_UPFACING | SURFACE_ROUGHNESS_DOWNFACING | SURFACE_ROUGHNESS_TREATED | ETCH_FACTOR | FINISHED_HEIGHT | WEIGHT | OTHER. If OTHER is used, `comment` is required. | 0-1        |
| comment                  | string                   | Additional description for requirements that differ from or supplement the enumerations.                                  | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the conductor requirement. Units here donâ€™t affect global ECAD units.                 | 0-n        |

Examples:
```xml
<Spec name="copper_foil2">
  <Conductor type="WEIGHT">
    <Property value="0.5" unit="OZ"/>
  </Conductor>
  <Conductor foilType="CU-E1"/>
  <Conductor type="SURFACE_ROUGHNESS_UPFACING">
    <Property value="3" unit="MICRON" refUnit="RMS"/>
  </Conductor>
</Spec>

<Spec name="coin_material">
  <Conductor material="COPPER"/>
</Spec>

<Spec name="conductive_ink">
  <Conductor material="SILVER_INK"/>
  <Conductor type="PRODUCT_NAME">
    <Property text="PE410"/>
  </Conductor>
</Spec>
```
