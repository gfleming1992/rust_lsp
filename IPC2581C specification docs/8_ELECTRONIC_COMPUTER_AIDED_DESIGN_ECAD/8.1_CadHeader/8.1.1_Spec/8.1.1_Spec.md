# 8.1.1 Spec

The `Spec` element holds specifications related to board/assembly characteristics and end-product capability, such as conductor widths/spacings, component spacing, impedance targets, compliance, dielectric/thermal properties, and dimensional constraints (e.g., V-cut, backdrill).

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                                | Occurrence |
|--------------------------|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| Spec                     | SpecType                 | Contains information that applies to board or assembly fabrication. May include values for conductor width/spacing, component spacing, impedance, etc.     | 0-n        |
| name                     | qualifiedNameType        | Unique name of the specification; indicates which product or scope in the file it pertains to.                                                             | 1-1        |
| SpecificationType        | ABSTRACT                 | Substitution group allowing specific specification types (Backdrill, Compliance, Conductor, Dielectric, General, Impedance, Technology, Temperature, etc.) | 0-n        |
| Xform                    | XformType                | Optional transform to reset origin, scale, mirror, or rotate the specified area.                                                                            | 0-1        |
| Location                 | LocationType             | Optional reference to a predefined image location where the specification applies.                                                                          | 0-1        |
| Outline                  | OutlineType              | Optional closed polygon(s) describing the area where the specification applies.                                                                             | 0-n        |
