# 8.1.1.15 EdgePlating

The `EdgePlating` specification defines copper plating on the boardâ€™s edge (cross section), including thickness, gaps, and surface finish.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                       | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| EdgePlating              | EdgePlatingType          | Identifies an edge plating specification instance.                                                                                                 | 0-n        |
| comment                  | string                   | Any comment(s) related to edge plating.                                                                                                            | 0-n        |
| PlatingThickness         | LengthPropertyType       | Required plating thickness; up to two instances allowed with constraintType MIN and/or MAX; otherwise nominal.                                     | 0-2        |
| PlatingGap               | LengthPropertyType       | Allowed gap in edge plating; up to two instances with constraintType MIN and/or MAX; otherwise nominal.                                           | 0-2        |
| SurfaceFinish            | SurfaceFinishType        | Enumerated finish per IPC-6012; if two provided, applied in order listed.                                                                          | 0-2        |

Example:
```xml
<Spec name="Edge_Plating_P1">
  <EdgePlating>
    <PlatingThickness value="4.0" unit="MICRON" plusTol="0.5" minusTol="0.5" tolPercent="false" constraintType="MIN"/>
    <PlatingThickness value="6.0" unit="MICRON" plusTol="0.5" minusTol="0.5" tolPercent="false" constraintType="MAX"/>
    <PlatingGap value="2.0" unit="MIL" constraintType="MAX"/>
    <SurfaceFinish type="G"/>
    <SurfaceFinish type="N"/>
  </EdgePlating>
</Spec>
```
