# 8.1.1.12 V_Cut

The `V_Cut` specification defines V-score (diamond wheel) cutting requirements on panels/boards, including angle, remaining thickness, and top-bottom offset.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                          | Occurrence |
|--------------------------|--------------------------|------------------------------------------------------------------------------------------------------|------------|
| V_Cut                    | V_CutType                | Identifies a V-cut specification instance.                                                           | 0-n        |
| type                     | vCutListType             | ANGLE | THICKNESS_REMAINING | OFFSET | OTHER. If OTHER is used, `comment` is required.              | 1-1        |
| comment                  | string                   | Additional description supplementing the enumerations.                                               | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the V-cut requirement.                                          | 0-n        |

Linking guidance: Draw the cut as a straight Line on a `layerFunction="V_CUT"` layer. Link to the Spec via SpecRef on the Layer or per LayerFeature:Set.

Example:
```xml
<Spec name="VCut_1">
  <V_Cut type="ANGLE">
    <Property value="90" unit="DEGREES" plusTol="5" minusTol="5" tolPercent="true"/>
  </V_Cut>
  <V_Cut type="THICKNESS_REMAINING">
    <Property value="0.5" unit="MM" plusTol="0.1" minusTol="0.1"/>
  </V_Cut>
  <V_Cut type="OFFSET">
    <Property value="0.0" unit="MM" plusTol="0.1" minusTol="0.1"/>
  </V_Cut>
</Spec>
```
