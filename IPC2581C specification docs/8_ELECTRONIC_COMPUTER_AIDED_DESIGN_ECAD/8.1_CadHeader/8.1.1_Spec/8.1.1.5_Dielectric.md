# 8.1.1.5 Dielectric

The `Dielectric` specification defines dielectric material properties and related process characteristics.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                              | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------|------------|
| Dielectric               | DielectricType           | Identifies a dielectric specification instance.                                                                            | 0-n        |
| type                     | dielectricListType       | DIELECTRIC_CONSTANT | LOSS_TANGENT | GLASS_STYLE | RESIN_CONTENT | Tg_DSC | Tg_DMA | Tg_TMA | Td | SLASH_IPC4101 | SLASH_IPC4103 | PRODUCT_NAME | OTHER. If OTHER is used, `comment` is required. | 1-1        |
| comment                  | string                   | Additional description for requirements that differ from or supplement the enumerations.                                  | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the dielectric requirement.                                                           | 0-n        |

Examples:
```xml
<Spec name="dielec4_15">
  <Dielectric type="DIELECTRIC_CONSTANT">
    <Property value="4.150" refValue="10.0e9" refUnit="Hz"/>
    <Property value="4.148" refValue="15.0e9" refUnit="Hz"/>
  </Dielectric>
  <Dielectric type="LOSS_TANGENT">
    <Property value="0.035" refValue="10.0e9" refUnit="Hz"/>
    <Property value="0.034" refValue="15.0e9" refUnit="Hz"/>
  </Dielectric>
  <Dielectric type="RESIN_CONTENT">
    <Property value="50" unit="PERCENT"/>
  </Dielectric>
  <Dielectric type="GLASS_STYLE">
    <Property value="1080"/>
  </Dielectric>
</Spec>

<Spec name="dielectric_layer5">
  <Dielectric type="PRODUCT_NAME">
    <Property text="I-Speed"/>
  </Dielectric>
  <Dielectric type="DIELECTRIC_CONSTANT">
    <Property value="3.64"/>
  </Dielectric>
  <Dielectric type="LOSS_TANGENT">
    <Property value="0.006"/>
  </Dielectric>
  <Dielectric type="Tg_DSC">
    <Property value="180" unit="CELCIUS" refUnit="MAX"/>
  </Dielectric>
  <Dielectric type="Td">
    <Property value="360" unit="CELCIUS"/>
  </Dielectric>
  <Dielectric type="SLASH_IPC4101">
    <Property value="98"/>
    <Property value="99"/>
    <Property value="101"/>
    <Property value="126"/>
  </Dielectric>
</Spec>
```
