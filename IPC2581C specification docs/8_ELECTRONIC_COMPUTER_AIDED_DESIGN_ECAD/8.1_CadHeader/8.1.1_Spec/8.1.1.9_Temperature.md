# 8.1.1.9 Temperature

The `Temperature` specification defines temperature-related capabilities/requirements to maintain physical properties for reliable performance.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                     | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------------------------------------|------------|
| Temperature              | TemperatureType          | Identifies a temperature specification instance.                                                                                 | 0-n        |
| type                     | temperatureListType      | THERMAL_DELAMINATION | EXPANSION_Z_AXIS | EXPANSION_X_Y_AXIS | OTHER. If OTHER is used, `comment` is required. | 1-1        |
| comment                  | string                   | Description supplementing/differing from enumerations.                                                                           | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to temperature impact characteristics.                                                         | 0-n        |

Example:
```xml
<Spec name="minimum_Td">
  <Temperature type="THERMAL_DELAMINATION">
    <Property value="180" unit="CELCIUS"/>
  </Temperature>
</Spec>
```
