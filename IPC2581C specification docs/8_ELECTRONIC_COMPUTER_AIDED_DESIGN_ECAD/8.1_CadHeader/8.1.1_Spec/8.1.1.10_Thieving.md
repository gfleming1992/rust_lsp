# 8.1.1.10 Thieving

The `Thieving` specification defines keep-in or keep-out constraints for applying copper thieving during PCB fabrication.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                 | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------|------------|
| Thieving                 | ThievingType             | Identifies a thieving specification instance.                                               | 0-n        |
| type                     | thievingListType         | KEEP_IN \| KEEP_OUT.                                                                          | 1-1        |
| comment                  | string                   | Description supplementing/differing from enumerations.                                      | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes; e.g., layer references.                                         | 0-n        |

Linking guidance: Define outline(s) on a layer with `layerFunction="THIEVING_KEEP_INOUT"`, then reference the `Spec` via a `SpecRef` on the Layer or on individual LayerFeature:Set entries.

Example:
```xml
<Spec name="Ethernet_POE_Connector_Thieving">
  <Thieving type="KEEP_OUT">
    <Property layerOrGroupRef="Layer1"/>
    <Property layerOrGroupRef="Layer2"/>
    <Property layerOrGroupRef="Layer3"/>
  </Thieving>
</Spec>
```
