# 8.1.1.14 SecondaryDrill

_TODO: add documentation._

The `SecondaryDrill` specification defines countersink/counterbore operations (secondary drilling), not electrical backdrilling.

| Attribute / Element Name | Attribute / Element Type   | Description                                                                                                            | Occurrence |
|--------------------------|----------------------------|------------------------------------------------------------------------------------------------------------------------|------------|
| SecondaryDrill           | SecondaryDrillType        | Identifies a secondary drill specification instance.                                                                    | 0-n        |
| type                     | SecondaryDrillListType    | MAJOR_DIAMETER | DEPTH | SIDE | ANGLE | OTHER. If OTHER is used, `comment` is required.                                   | 1-1        |
| comment                  | string                     | Description supplementing/differing from enumerations.                                                                  | 0-1        |

Examples:
```xml
<Spec name="Countersink_1">
  <SecondaryDrill type="MAJOR_DIAMETER">
    <Property value="0.5" unit="MM" plusTol="0" minusTol="5" tolPercent="true"/>
  </SecondaryDrill>
  <SecondaryDrill type="ANGLE">
    <Property value="90" unit="DEGREES"/>
  </SecondaryDrill>
  <SecondaryDrill type="SIDE">
    <Property text="TOP"/>
  </SecondaryDrill>
</Spec>

<Spec name="Counterbore_1">
  <SecondaryDrill type="MAJOR_DIAMETER">
    <Property value="0.5" unit="MM" plusTol="0" minusTol="5" tolPercent="true"/>
  </SecondaryDrill>
  <SecondaryDrill type="DEPTH">
    <Property value="0.3" unit="MM"/>
  </SecondaryDrill>
  <SecondaryDrill type="SIDE">
    <Property text="BOTTOM"/>
  </SecondaryDrill>
</Spec>
```
