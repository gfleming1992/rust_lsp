# 8.1.1.2 Backdrill

The `Backdrill` specification defines backdrilling requirements for plated-through holes or vias to remove excess plating without disturbing the interconnection structure.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                              | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------|------------|
| Backdrill                | BackdrillType            | Identifies a backdrill specification instance.                                                                            | 0-n        |
| type                     | backdrillListType        | START_LAYER | MUST_NOT_CUT_LAYER | MAX_STUB_LENGTH | OTHER. If OTHER is used, `comment` is required.                                      | 1-1        |
| comment                  | string                   | Additional description for requirements that differ from or supplement the enumerations.                                  | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the backdrill requirement; units apply only to this Spec, not global ECAD units.     | 0-n        |

Example:
```xml
<Spec name="backdrill_1">
  <Backdrill type="START_LAYER">
    <Property layerOrGroupRef="L12"/>
  </Backdrill>
  <Backdrill type="MUST_NOT_CUT_LAYER">
    <Property layerOrGroupRef="L8"/>
  </Backdrill>
  <Backdrill type="MAX_STUB_LENGTH">
    <Property value="0.5" unit="MM" plusTol="0" minusTol="5" tolPercent="true"/>
  </Backdrill>
</Spec>
```
