# 8.1.1.8 Technology

The `Technology` specification defines the mounting platform technology of the product (e.g., rigid, flex) to influence related material and structural specs.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                 | Occurrence |
|--------------------------|--------------------------|-------------------------------------------------------------------------------------------------------------|------------|
| Technology               | TechnologyType           | Identifies a technology specification instance.                                                              | 0-n        |
| type                     | technologyListType       | RIGID | RIGID_FLEX | FLEX | HDI | EMBEDDED_COMPONENT | OTHER. If OTHER is used, `comment` is required. | 1-1        |
| comment                  | string                   | Additional description supplementing the enumerations.                                                       | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the technology requirement.                                            | 0-n        |

Example:
```xml
<Spec name="board_technology">
  <Technology type="RIGID"/>
</Spec>
```
