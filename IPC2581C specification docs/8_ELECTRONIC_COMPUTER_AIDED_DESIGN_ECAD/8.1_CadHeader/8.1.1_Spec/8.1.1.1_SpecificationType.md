# 8.1.1.1 SpecificationType

The `SpecificationType` is an abstract substitution group that allows different specification functions to be defined within a `Spec` element. Possible specification types include Backdrill, Compliance, Conductor, Dielectric, EdgeChamfer, EdgePlating, Flex, General, Impedance, Loss, SecondaryDrill, SurfaceFinish, Technology, Temperature, Thieving, Tool, and V_Cut.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                                | Occurrence |
|--------------------------|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| SpecificationType        | ABSTRACT                 | Substitution group for specific specification elements that focus descriptions to where they apply.                                                       | 0-n        |
| Backdrill                | BackdrillType            | Defines backdrill requirements for plated-through holes or vias.                                                                                           | 0-n        |
| Compliance               | ComplianceType           | Defines regulatory compliance requirements (e.g., RoHS, REACH, WEEE).                                                                                     | 0-n        |
| Conductor                | ConductorType            | Defines conductor characteristics (material, surface, etch factor, finished height, weight, etc.).                                                        | 0-n        |
| Dielectric               | DielectricType           | Defines dielectric properties and process characteristics.                                                                                                 | 0-n        |
| EdgeChamfer              | EdgeChamferType          | Defines board-edge chamfer characteristics.                                                                                                                | 0-n        |
| EdgePlating              | EdgePlatingType          | Defines copper plating on the board edge cross-section.                                                                                                    | 0-n        |
| Flex                     | FlexType                 | Defines properties associated with flex zones.                                                                                                             | 0-n        |
| General                  | GeneralType              | Defines general requirements (electrical, thermal, material, instructions, standards).                                                                     | 0-n        |
| Impedance                | ImpedanceType            | Defines impedance requirements and transmission structures.                                                                                                | 0-n        |
| Loss                     | LossType                 | Defines electrical loss requirements.                                                                                                                      | 0-n        |
| SecondaryDrill           | SecondaryDrillType       | Defines countersink/counterbore requirements.                                                                                                              | 0-n        |
| SurfaceFinish            | SurfaceFinishType        | Defines surface finish materials and product options.                                                                                                      | 0-n        |
| Technology               | TechnologyType           | Defines the mounting platform technology (rigid, flex, HDI, etc.).                                                                                         | 0-n        |
| Temperature              | TemperatureType          | Defines temperature-related material/assembly limits.                                                                                                      | 0-n        |
| Thieving                 | ThievingType             | Defines thieving keep-in/keep-out constraints.                                                                                                             | 0-n        |
| Tool                     | ToolType                 | Defines tooling requirements.                                                                                                                               | 0-n        |
| V_Cut                    | V_CutType                | Defines V-cut scoring requirements.                                                                                                                         | 0-n        |
