# 8.1.1.3 Compliance

The `Compliance` specification defines legal/regulatory requirements (e.g., RoHS, WEEE) for materials or the final product.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                              | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------|------------|
| Compliance               | ComplianceType           | Identifies a compliance specification instance.                                                                           | 0-n        |
| type                     | complianceListType       | ROHS | CONFLICT_MINERALS | WEEE | REACH | HALOGEN_FREE | OTHER. If OTHER is used, `comment` is required.                                     | 1-1        |
| comment                  | string                   | Additional description for requirements that differ from or supplement the enumerations.                                  | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to compliance; units apply only to this Spec, not global ECAD units.                    | 0-n        |

Example:
```xml
<Spec name="compliances">
  <Compliance type="ROHS"/>
  <Compliance type="HALOGEN_FREE"/>
</Spec>
```
