# 8.1.1.16 SurfaceFinish

The `SurfaceFinish` specification defines the finish material(s) applied to conductive surfaces. It can be referenced by StackupLayer (for COATINGCOND/COATINGNONCOND), as a child of EdgePlating, or by a conductive Model via SpecRef.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                    | Occurrence |
|--------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------|------------|
| Finish                   | FinishType               | Indicates a surface finish material.                                                                            | 0-1        |
| type                     | surfaceFinishType        | Enumerated per IPC-6012 (e.g., S, T, X, TLU, G, GS, N, NB, OSP, HT_OSP, ENIG-G, ENEPIG, DIG, IAg, ISn, C, OTHER). | 0-1        |
| comment                  | string                   | Description for OTHER or any additional finish info.                                                            | 1-1        |
| Product                  | ProductType              | Nested elements providing specific brand names of surface finishes.                                             | 0-n        |
| name                     | string                   | Brand/product name.                                                                                             | 1-1        |
| criteria                 | productCriteriaType      | ALLOWED | SUGGESTED | PREFERRED | REQUIRED | CHOSEN.                                                                  | 1-1        |

Examples:
```xml
<Finish type="HT_OSP">
  <Product name="Entek HT" criteria="ALLOWED"/>
  <Product name="Gilcoat SMD F2(LX)" criteria="ALLOWED"/>
</Finish>

<Finish type="ENIG-G"/>
<Finish type="OTHER" comment="no finish required"/>
```
