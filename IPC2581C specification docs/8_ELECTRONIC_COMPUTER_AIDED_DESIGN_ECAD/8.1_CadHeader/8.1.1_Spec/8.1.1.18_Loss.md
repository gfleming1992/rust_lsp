# 8.1.1.18 Loss

The `Loss` specification defines electrical loss requirements and can be referenced by LayerFeature:Set, LogicalNet, StackupGroup, or Stackup.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                  | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------|------------|
| Loss                     | LossType                 | Identifies a loss specification instance.                                                                     | 0-n        |
| type                     | lossListType             | ATTENUATION | IMPEDANCE | INSERTION | POWER | SIGNAL | VOLTAGE.                                        | 1-1        |
| comment                  | string                   | Comment describing OTHER or any additional information.                                                       | 0-1        |
| Property                 | PropertyType             | Numerical attributes relating to the loss requirement (units local to Spec).                                  | 0-n        |

Examples:
```xml
<Spec name="attenuation_vs_frequency">
  <Loss type="ATTENUATION">
    <Property value="3" unit="dB/INCH" refValue="10e9" refUnit="Hz"/>
    <Property value="7" unit="dB/INCH" refValue="15e9" refUnit="Hz"/>
  </Loss>
</Spec>

<Spec name="max_insertion_loss">
  <Loss type="INSERTION">
    <Property value="3" unit="dB" refUnit="MAX"/>
  </Loss>
</Spec>
```
