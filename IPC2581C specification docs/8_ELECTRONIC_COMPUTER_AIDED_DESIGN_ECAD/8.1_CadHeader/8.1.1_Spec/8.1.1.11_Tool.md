# 8.1.1.11 Tool

The `Tool` specification defines tool types and properties required for fabrication or assembly (e.g., drill size, bit angle, torque).

| Attribute / Element Name | Attribute / Element Type   | Description                                                                                                         | Occurrence |
|--------------------------|----------------------------|---------------------------------------------------------------------------------------------------------------------|------------|
| Tool                     | ToolType                   | Identifies a tool specification instance.                                                                            | 0-n        |
| type                     | toolListType               | CARBIDE | ROUTER | LASER | FLATNOSE | EXTENSION | V_CUTTER | SCREWDRIVER | WRENCH | OTHER. If OTHER, `comment` required. | 1-1        |
| toolProperty             | toolPropertyListType       | DRILL_SIZE | FINISHED_SIZE | BIT_ANGLE | TORQUE | HEX_NUT_SIZE | PHILLIPS_HEAD | FLAT_HEAD | TORX_HEAD | ALLEN_HEAD | OTHER. If OTHER, `comment` required. | 1-1        |
| comment                  | string                     | Additional description supplementing the enumerations.                                                               | 0-1        |
| Property                 | PropertyType               | Numerical/text attributes defining the property values (units local to Spec).                                       | 0-n        |

Examples:
```xml
<Spec name="drill_1">
  <Tool type="CARBIDE" toolProperty="DRILL_SIZE">
    <Property text="B" unit="GAUGE"/>
  </Tool>
  <Tool type="CARBIDE" toolProperty="FINISHED_SIZE">
    <Property value="0.5" unit="MM" plusTol="0.05" minusTol="0.05"/>
  </Tool>
</Spec>

<Spec name="v_cutter_tool">
  <Tool type="V_CUTTER" toolProperty="BIT_ANGLE">
    <Property value="90" unit="DEGREES"/>
  </Tool>
</Spec>

<Spec name="screw_torque">
  <Tool type="SCREWDRIVER" toolProperty="FLAT_HEAD"/>
  <Tool type="SCREWDRIVER" toolProperty="TORQUE">
    <Property value="7.5" unit="IN-LB"/>
  </Tool>
</Spec>

<Spec name="nut_torque">
  <Tool type="WRENCH" toolProperty="HEX_NUT_SIZE">
    <Property text="M2"/>
  </Tool>
  <Tool type="WRENCH" toolProperty="TORQUE">
    <Property value="6.5" unit="IN-LB" refUnit="MIN"/>
    <Property value="7.0" unit="IN-LB" refUnit="MAX"/>
  </Tool>
</Spec>
```
