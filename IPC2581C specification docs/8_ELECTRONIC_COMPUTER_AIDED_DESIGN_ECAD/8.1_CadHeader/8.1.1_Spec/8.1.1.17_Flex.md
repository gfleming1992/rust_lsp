# 8.1.1.17 Flex

The `Flex` specification defines properties associated with flex StackupZones; referenced via SpecRef from a StackupZone flex element.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                           | Occurrence |
|--------------------------|--------------------------|-------------------------------------------------------------------------------------------------------|------------|
| Flex                     | FlexType                 | Identifies a flex specification instance.                                                              | 0-n        |
| type                     | flexListType             | ADHESIVE_SQUEEZE_OUT | DIELECTRIC_SQUEEZE_OUT | STRESS_RELIEF_FILLET_WIDTH | OTHER.     | 1-1        |
| comment                  | string                   | Description supplementing/differing from enumerations.                                                 | 0-1        |
| Property                 | PropertyType             | Numerical/text attributes related to the flex characteristic.                                          | 0-n        |

Examples:
```xml
<Spec name="SqueezeOut">
  <Flex type="DIELECTRIC_SQUEEZE_OUT">
    <Property value="1.2" unit="MM" refUnit="MAX"/>
  </Flex>
</Spec>

<Spec name="EpoxyAdhesiveStressRelief">
  <Flex type="STRESS_RELIEF_FILLET_WIDTH">
    <Property value="1" unit="MM" refUnit="MIN"/>
    <Property value="2" unit="MM" refUnit="MAX"/>
  </Flex>
</Spec>
```
