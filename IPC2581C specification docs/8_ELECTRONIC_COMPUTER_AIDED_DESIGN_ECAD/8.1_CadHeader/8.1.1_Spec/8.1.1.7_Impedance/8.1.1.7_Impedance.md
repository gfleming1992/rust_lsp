# 8.1.1.7 Impedance

The Impedance specification defines impedance characteristics for the product, including the required value, tolerance, and the transmission/structure details needed to meet it.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                        | Occurrence |
|--------------------------|--------------------------|------------------------------------------------------------------------------------------------------------------------------------|------------|
| Impedance                | ImpedanceType            | Specification describing impedance, structure, and optionally tolerance.                                                           | 0-n        |
| value                    | double                   | Impedance value in ohms (e.g., 10e3 = 10 kÎ©).                                                                                      | 1-1        |
| tolPlus                  | nonNegativeDouble        | Upper tolerance for the impedance value.                                                                                           | 0-1        |
| tolMinus                 | nonNegativeDouble        | Lower tolerance for the impedance value.                                                                                           | 0-1        |
| tolPercent               | boolean                  | If true, tolPlus/tolMinus are percentages; otherwise absolute. Default false.                                                      | 0-1        |
| TransmissionType         | ABSTRACT                 | One of SingleEnded, EdgeCoupled, BroadsideCoupled, CoplanarWaveguide; omitted only for LogicalNet-wide SpecRef.                    | 0-1        |
| comment                  | string                   | Optional explanatory comment.                                                                                                      | 0-1        |

See subsections for each TransmissionType, common child elements, and SpecRef instantiation rules.
