# 8.1.1.7.2 EdgeCoupled

A permitted substitution of TransmissionType as a child of `Impedance`, to identify an edge-coupled differential transmission type.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                       | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------|------------|
| EdgeCoupled              | EdgeCoupledType          | Edge-coupled impedance specification.                                                             | 0-1        |
| structure                | structureListType        | STRIPLINE_SYMMETRIC | STRIPLINE_ASYMMETRIC | STRIPLINE_PLANE_LESS | MICROSTRIP_EMBEDDED | MICROSTRIP_NO_MASK | MICROSTRIP_MASK_COVERED | MICROSTRIP_DUAL_MASK_COVERED | 1-1 |
| LineWidth                | LengthPropertyType       | Trace width, unit of measure, and tolerance.                                                      | 1-1        |
| LineGap                  | ABSTRACT                 | Spacing (edge-to-edge) or Pitch (center-to-center).                                               | 1-1        |
| RefPlane                 | RefPlaneType             | 0â€“2 reference planes.                                                                             | 0-1        |

Example:
```xml
<Impedance value="100" tolPlus="10" tolMinus="10" tolPercent="true">
  <EdgeCoupled structure="STRIPLINE_SYMMETRIC">
    <LineWidth value="300" unit="MICRON" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <Pitch value="200" unit="MICRON" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <RefPlane layerOrGroupRef="L8"/>
    <RefPlane layerOrGroupRef="L10"/>
  </EdgeCoupled>
</Impedance>
```
