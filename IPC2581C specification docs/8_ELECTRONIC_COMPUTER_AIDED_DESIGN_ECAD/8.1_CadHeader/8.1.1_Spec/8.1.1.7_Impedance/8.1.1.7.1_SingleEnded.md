# 8.1.1.7.1 SingleEnded

A permitted substitution of TransmissionType as a child of `Impedance`, to identify a single-ended transmission type.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                       | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------|------------|
| SingleEnded              | SingleEndedType          | Single-ended impedance specification.                                                             | 0-1        |
| structure                | structureListType        | STRIPLINE_SYMMETRIC | STRIPLINE_ASYMMETRIC | STRIPLINE_PLANE_LESS | MICROSTRIP_EMBEDDED | MICROSTRIP_NO_MASK | MICROSTRIP_MASK_COVERED | MICROSTRIP_DUAL_MASK_COVERED | 1-1 |
| LineWidth                | LengthPropertyType       | Trace width, unit of measure, and tolerance.                                                      | 1-1        |
| RefPlane                 | RefPlaneType             | 0â€“2 reference planes.                                                                             | 0-1        |

Example:
```xml
<Impedance value="50" tolPlus="10" tolMinus="10" tolPercent="true">
  <SingleEnded structure="MICROSTRIP_MASK_COVERED">
    <LineWidth value="0.3" unit="MM"/>
    <RefPlane layerOrGroupRef="L2"/>
  </SingleEnded>
</Impedance>
```
