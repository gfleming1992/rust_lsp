# 8.1.1.7.3 BroadsideCoupled

A permitted substitution of TransmissionType as a child of `Impedance`, to identify a broadside-coupled differential transmission type.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                       | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------|------------|
| BroadsideCoupled         | BroadsideCoupledType     | Broadside-coupled impedance specification.                                                        | 0-1        |
| structure                | broadsideListType        | STRIPLINE_SYMMETRIC | STRIPLINE_ASYMMETRIC | STRIPLINE_PLANE_LESS                                  | 1-1        |
| LineWidth                | LengthPropertyType       | Trace width, unit of measure, and tolerance of the coupled pair.                                  | 1-1        |
| Offset                   | LengthPropertyType       | Horizontal offset between the pair’s corresponding edges; can be zero with tolerance.             | 0-1        |
| PairLayerRef             | RefPlaneType             | Reference to the layer occupied by the 2nd net (the current layer is the 1st net).                | 1-1        |
| RefPlane                 | RefPlaneType             | 0–2 reference planes.                                                                             | 0-2        |

Example:
```xml
<Impedance value="75" tolPlus="5" tolMinus="5" tolPercent="false">
  <BroadsideCoupled structure="STRIPLINE_ASYMMETRIC">
    <LineWidth value="25" unit="MIL" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <Spacing value="15" unit="MIL" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <RefPlane layerOrGroupRef="L5"/>
    <RefPlane layerOrGroupRef="L7"/>
  </BroadsideCoupled>
</Impedance>
```
