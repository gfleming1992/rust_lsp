# 8.1.1.7.4 CoplanarWaveguide

A permitted substitution of TransmissionType as a child of `Impedance`, to identify a coplanar waveguide transmission type.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                       | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------|------------|
| CoplanarWaveguide        | CoplanarWaveguideType    | Coplanar waveguide impedance specification.                                                       | 0-1        |
| structure                | coplanarListType         | COPLANAR_WAVEGUIDE_STRIPLINE | COPLANAR_WAVEGUIDE_EMBEDDED | COPLANAR_WAVEGUIDE_NO_MASK | COPLANAR_WAVEGUIDE_MASK_COVERED | COPLANAR_WAVEGUIDE_DUAL_MASKED_COVERED | OTHER | 1-1 |
| LineWidth                | LengthPropertyType       | Trace width, unit of measure, and tolerance.                                                      | 1-1        |
| LineGap                  | ABSTRACT                 | Spacing or Pitch substitution.                                                                     | 1-1        |
| CoplanarGroundSpacing    | LengthPropertyType       | Distance from outer edge of coupled pair to same-layer ground edge.                                | 1-1        |
| RefPlane                 | RefPlaneType             | 0â€“2 optional reference planes.                                                                     | 0-1        |

Example:
```xml
<Impedance value="75" tolPlus="5" tolMinus="5" tolPercent="false">
  <CoplanarWaveguide structure="COPLANAR_WAVEGUIDE_EMBEDDED">
    <LineWidth value="25" unit="MIL" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <Spacing value="15" unit="MIL" tolPlus="10" tolMinus="10" tolPercent="true"/>
    <CoplanarGroundSpacing value="5" unit="MIL" tolPlus="10" tolMinus="10" tolPercent="true"/>
  </CoplanarWaveguide>
</Impedance>
```
