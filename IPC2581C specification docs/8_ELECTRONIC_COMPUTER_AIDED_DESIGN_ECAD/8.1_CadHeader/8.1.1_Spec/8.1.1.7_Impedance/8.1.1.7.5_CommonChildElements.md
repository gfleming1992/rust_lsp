# 8.1.1.7.5 Transmission Type Common Child Elements

These elements are shared across transmission types to define physical dimensions and references.

| Element Name              | Element Type         | Description                                                                                 |
|---------------------------|----------------------|---------------------------------------------------------------------------------------------|
| LineWidth, Spacing, Pitch, Offset, CoplanarGroundSpacing | LengthPropertyType | Describes a length with value, unit, and tolerance.                                          |

LengthPropertyType attributes:
- value (double) – The length value
- unit (lengthUnitType) – MM | MICRON | MIL | INCH
- tolPlus (nonNegativeDouble) – Upper tolerance
- tolMinus (nonNegativeDouble) – Lower tolerance
- tolPercent (boolean) – If true, tolerances are percentage; default false

RefPlane and PairLayerRef:

| Element Name   | Element Type | Description                                           |
|----------------|--------------|-------------------------------------------------------|
| RefPlane, PairLayerRef | RefPlaneType  | References a StackupLayer or StackupGroup by name.   |

RefPlaneType attributes:
- layerOrGroupRef (qualifiedNameType) – Unique StackupLayer or StackupGroup name
