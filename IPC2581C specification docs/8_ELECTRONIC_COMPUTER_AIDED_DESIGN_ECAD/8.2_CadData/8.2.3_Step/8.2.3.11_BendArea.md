# 8.2.3.11 BendArea
 
An embedded element that defines either a single bend, or one of a sequence of bends to a flex circuit. 
 
## Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| BendArea | BendAreaType | A nested element containing the different layer information represented in the CAD data base which describes the electronic assembly to be manufactured. | 0-n |
| name | qualifiedNameType | The identification of the bend area. Each bend area must be a unique name. | 1-1 |
| sequenceNumber | integer | An integer for which order in sequence a bend should take place in reference to other bends. | 0-1 |
| comment | string | A description of the Bend Area as provided in the original CAD data | 0-1 |
| CircularBend | CircularBendType | An element that contains the bending information | 1-1 |
| Outline | OutlineType | A nested element that defines the physical outline of the bend area as viewed from the top. | 1-1 |


## 8.2.3.11.1 CircularBend
 
The bend type description for a bend that is a circular form. 
 
### Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| CircularBend | CircularBendType | The bend type description for a bend that is a circular form | 0-1 |
| innerSide | bendSideType | Side name for the inner side of the surface that is bending as viewed from the Top of the design. TOP indicate the outermost extents of the bend move upward. BOTTOM indicates the outermost extents of the bend move downward. | 1-1 |
| innerRadius | double | The radius of the bend | 1-1 |
| innerAngle | angleType | The final angle of the bend from the opposite side extents of the bend area outline and parallel to the bend line. | 0-1 |
| BendLine | LineType | The apex axis of the bend position on the inside surface of the bend. | 1-1 |

                     <LineDesc lineEnd="ROUND" lineWidth="0.000000"/>

 </BendArea>
