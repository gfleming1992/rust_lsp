# 8.2.3.12 StackupZone

A nested element defining stackup zones in a rigid flex design. The zones are created by planes ‘slicing’ vertically, 
that is perpendicular, to the plane of the board. They occur at the boundary of a stackup change between rigid and 
flex sections of the board. 
 
## Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| StackupZone | StackupZoneType | The definition of the area and properties of a stackup zone | 1-n |
| name | qualifiedNameType | If more than one stack up zone, it is recommended to have a unique name for each stackup zone. Duplicate names are not allowed. | 0-1 |
| stackupRef | qualifiedNameType | A reference to the stackup layer structure that is contained within this zone structure | 1-1 |
| Profile | ContourType | The profile of the stackup zone. | 1-1 |
| SpecRef | SpecRefType | A reference to flex or other properties of the zone | 0-1 |


### Example

```xml
<StackupZone name=“Zone_1"> 
         <Profile> 
               <Polygon> 
 <PolyBegin x= “-.250” y=“-.250”/> 
 <PolyStepSegment x= “2.750 y= “.-250”/> 
 <PolyStepSegment x= “2.750” y= “.2.750”/> 
 <PolyStepSegment x= “-.250” y= “.2.750”/> 
 <PolyStepSegment x= “-.250” y= “-250”/> 
               </Polygon> 
             </Profile> 
          <StackupRef name=“Primary”/> 
</StackupZone> 
 
<StackupZone name=“Zone_2"> 
         <Profile> 
               <Polygon> 
 <PolyBegin x= “2.750” y=“0.000”/> 
 <PolyStepSegment x= “5.750 y= “0.000”/> 
 <PolyStepSegment x= “5.750” y= “2.000”/> 
 <PolyStepSegment x= “2.750” y= “2.000”/> 
 <PolyStepSegment x= “2.750” y= “0.000”/> 
               </Polygon> 
             </Profile> 
          <ZoneLayer layerRef=”CoverLay” InsetSize=”0.010”> 
           <StackupRef name=“Flex”/> 
</StackupZone> 
<StackupZone name=“Zone_1"> 
         <Profile> 
               <Polygon> 
 <PolyBegin x= “5.750” y=“-.250”/> 
 <PolyStepSegment x= “7.750 y= “.-250”/>  
 <PolyStepSegment x= “7.750” y= “.2.750”/> 
 <PolyStepSegment x= “5.750” y= “.2.750”/> 
 <PolyStepSegment x= “5.750” y= “-250”/> 
               </Polygon> 
             </Profile> 
          <StackupRef name=“Primary”/> 
</StackupZone>  
```

## 8.2.3.12.1 ZoneLayer
 
Layer specific attributes for a stackup layer in a zone. 
 
### Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| ZoneLayer | ZoneLayerType | Layer specific attributes for a stackup layer in a zone | 0-1 |
| layerOrGroupRef | qualifiedNameType | A reference to a layer in the stackup layer structure that is contained within this zone structure. | 1-1 |
| InsetSize | Double | The dimension of the inset oversize from the coverlay or other flexible layer into the rigid zone.   InsetSize is intended to be used for enlarging a flexible zone where it is inserted into an adjacent rigid zone. | 1-1 |
| StackupZoneRef | StackupZoneRefType | The names of adjacent zones to which the inset zone applies.  This allows different inset values for different adjacent rigid zones.  If no stackup zone references are specified then the InsetSize attribute is applied to all adjacent rigid zones for the specified flexible zone. | 0-n |
