## 8.2.3.15.1 Criteria

The purpose of this element is to define the measurement criteria that was used for DFx Analysis and to list the 
measurements that violate the defined criteria. The criteria may range from any design for manufacturing validation 
rules, process rules, note details, or instruction details. The Property element is used as the descriptor for the values. 
 
### Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| Criteria | CriteriaType | An element consisting of the definition of a particular requirement of the product, and a list of all the measurements made in the design for manufacturing analysis that do not meet that requirement.  | 1-1 |
| name | qualifiedNameType | A descriptive name that describes the specific measurement or detail | 1-1 |
| measurementMode | measurementModeType | The mode of measurement depending on the characteristic being evaluated, indicated as an enumerated string using: DISTANCE | AREA | RESISTANCE | CAPACITANCE | IMPEDANCE | INDUCTANCE | VOLTAGE_DC | VOLTAGE_AC | FREQUENCY | CURRENT | POWER | PERCENTAGE | SIZE | NONE | TEXT | TOLERANCE | OTHER | 1-1 |
| comment | string | Any comment to assist in the definition or description of the unit details. Mandatory when measurementMode value is the enumerated type OTHER | 0-1 |
| Property | PropertyType | An element that provides the definition of the measurement details as the base definition  | 1-1 |
| DfxMeasurement | DfxMeasurementType | An element consisting of a measurement made in the design which violates its parent Criteria | 1-n |
