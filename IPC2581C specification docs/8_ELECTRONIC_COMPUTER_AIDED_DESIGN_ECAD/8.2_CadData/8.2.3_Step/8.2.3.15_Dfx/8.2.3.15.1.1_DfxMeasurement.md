## 8.2.3.15.1.1 DfxMeasurement

The DfxMeasurement element describes details of a measurement that violates its parent Criteria. The 
measurement may be made on a one layer or may span more than one layer. 

### Attributes

| Attribute /  Element Name | Attribute /  Element Type | Description | Occurrence |
|---|---|---|---|
| DfxMeasurement | DfxMeasurementType | An embedded element that provides the specific details of the measurements that are made on individual layers or product to determine design for manufacturing characteristics. | 1-n |
| id | qualifiedNameType | A unique identifier that can be referenced by a DfxResponse | 1-1 |
| severity | string | A user defined indication of the seriousness of the violation. | 0-1 |
| comment | string | Any comment to assist in the definition or description of the unit details, required when OTHER is used. | 0-1 |
| Property | PropertyType | An element used in defining the actual value of the measurement to which the Critera property characteristics is compared to. | 1-1 |
| MeasurementPoint | MeasurementPointType | An element used in defining the specific point, or points, where the measurement was taken, with reference to a given layer in tre design.   | 1-n |
| DfxDetails | DfxDetailsType | An optional nested element containg any details pertaining to the measurement, such as a location, marker, or feature description, or reference(s) to any design object(s) or external data. See section 8.2.3.15.2.1 DfxDetails | 0-1 |
