# 6.4 ChangeRec

The `ChangeRec` element provides configuration management information for changes made to the product represented by the IPC-2581 data file. Each `ChangeRec` entry is recorded by the date and identifies the initiating person, effectivity, and description of the change. Approval records may be nested to capture who approved the changes.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                          | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------|------------|
| ChangeRec                | ChangeRecType            | An element required to manage configuration of the changes made to the product during development and final configuration in the field. | 0-n        |
| datetime                 | dateTime                 | The timestamp recorded when the change was made to the file.                                                                           | 1-1        |
| personRef                | qualifiedNameType        | Reference to the `name` of the person listed in the `LogisticHeader` who initiated the change.                                          | 1-1        |
| application              | string                   | The effectivity of the change indicating when it becomes active (e.g., after so many completed units).                                | 1-1        |
| change                   | string                   | A short description of the change.                                                                                                    | 1-1        |
| Approval                 | ApprovalType             | A nested element that signifies who approved the suggested change.                                                                     | 0-n        |

Nested **Approval** element attributes:

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                          | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------------------|------------|
| datetime                 | dateTime                 | The timestamp recorded when the change was approved.                                                                                  | 1-1        |
| personRef                | qualifiedNameType        | Reference to the `name` of the person listed in the `LogisticHeader` who approved the change.                                         | 1-1        |

**Example:**
```xml
<ChangeRec datetime="2010-06-12T13:20:00" personRef="Harry Jones" application="The change is to be implemented immediately with all produced items reworked" change="Replace the chip resistors with new part number 34-67-95, sixteen places">
    <Approval datetime="2010-06-13T10:32:30" personRef="John Smithy"/>
</ChangeRec>
```
