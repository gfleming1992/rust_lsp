# 7.1 BOM Header

The `BomHeader` element contains identification and logistical information for a Bill of Materials (BOM) in an IPC-2581 file. It is a mandatory nested element of `Bom`, defining context for component grouping and assembly data.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                                                                               | Occurrence |
|--------------------------|--------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| BomHeader                | BomHeaderType            | A nested element containing identification and logistical information about the BOM.                                                                                                                       | 1-1        |
| assembly                 | string                   | Electronic assembly name as parsed from the BOM source.                                                                                                                                                    | 1-1        |
| revision                 | string                   | Revision identifier as parsed from the BOM source.                                                                                                                                                         | 1-1        |
| affecting                | boolean                  | Indicates whether this BOM was used in the assembly merge process ("true" or "false"). Only one BOM per job may be the affecting BOM.                                                                  | 0-1        |
| stepRef                  | string                   | Identification of specific steps used to describe `BomItem` entries within a material or component category.                                                                                               | 0-n        |

**Example:**
```xml
<Bom name="TestBoard1">
    <BomHeader assembly="Karens Design" revision="Prototype" affecting="true" stepRef="KarensBoard"/>
    <!-- ... other BomItem entries ... -->
</Bom>
```
