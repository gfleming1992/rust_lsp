# 7.2.1 BomDes

The Bill of Material Designations (`BomDes`) identify items in the BOM that need specific designations based on their characteristics. There are five groups of designations:

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                        | Occurrence |
|--------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| BomDes                   | BomDescType              | Substitution group that permits designation of BOM items according to the category identified by the LayerFunction.                                 | 0-n        |
| DocDes                   | DocDesType               | Designation for documents contained in the BOM.                                                                                                    | 0-n        |
| FindDes                  | FindDesType              | Designation for mechanical parts (Find Numbers) with no electrical connectivity; often groups of identical parts.                                    | 0-n        |
| MatDes                   | MatDesType               | Designation for materials contained in the BOM.                                                                                                    | 0-n        |
| RefDes                   | RefDesType               | Designation for components contained in the BOM; includes any part with electrical connectivity via pins/pads.                                     | 0-n        |
| ToolDes                  | ToolDesType              | Designation for tools or fixtures contained in the BOM.                                                                                             | 0-n        |

**Example:**
```xml
<Bom name="TestBoard1">
  <BomHeader assembly="DesignA" revision="Proto"/>
  <BomItem OEMDesignNumberRef="U1" quantity="1" category="ELECTRICAL">
    <DocDes name="Doc1" layerRef="DOCUMENT_LAYER"/>
    <FindDes number="100" layerRef="MECH_LAYER"/>
    <MatDes name="Mat1" layerRef="MATERIAL_LAYER"/>
    <RefDes name="R1" packageRef="ResistorPkg" populate="true"/>
    <ToolDes name="ToolA" layerRef="TOOL_LAYER"/>
  </BomItem>
</Bom>
```
