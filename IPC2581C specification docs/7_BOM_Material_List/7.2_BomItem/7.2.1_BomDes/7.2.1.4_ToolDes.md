# 7.2.1.4 ToolDes

The `ToolDes` element is a BOM item designation used to identify tools or fixtures contained in the BOM and referenced in the IPC-2581 file.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                      | Occurrence |
|--------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------------------------|------------|
| ToolDes                  | ToolDesType              | A BOM item designation for tools or fixtures contained in the BOM.                                                                | 0-n        |
| name                     | qualifiedNameType        | A unique name assigned as a designation for tools or fixtures (combination of letters and numbers).                             | 1-1        |
| layerRef                 | qualifiedNameType        | Reference to a layerFunction identifying the layer that describes the tools or fixtures in the CAD data.                        | 0-1        |

**Example:**
```xml
<ToolDes name="Tool1" layerRef="TOOL_LAYER"/>
```
