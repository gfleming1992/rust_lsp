# 7.2.1.5 FindDes

The `FindDes` element is a BOM item designation used to identify mechanical part numbers (Find Numbers) contained in the BOM and referenced in the IPC-2581 file. Find Numbers often represent collections of identical parts.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                                        | Occurrence |
|--------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|------------|
| FindDes                  | FindDesType              | A BOM item designation for mechanical parts with no electrical connectivity to the PCB.                                                            | 0-n        |
| number                   | positiveInteger          | A unique positive integer assigned as the Find Number.                                                                                            | 0-n        |
| layerRef                 | qualifiedNameType        | Reference to a layerFunction identifying where the mechanical parts are described in CAD data.                                                     | 0-1        |

**Example:**
```xml
<FindDes number="100" layerRef="MECHANICAL_LAYER"/>
```
