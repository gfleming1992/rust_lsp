# 7.2.1.3 RefDes

The `RefDes` element is a BOM item designation used to identify components contained in the BOM and referenced in the IPC-2581 file.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                     | Occurrence |
|--------------------------|--------------------------|---------------------------------------------------------------------------------------------------------------------------------|------------|
| RefDes                   | RefDesType               | A nested element containing the reference designator strings for individual parts, matching the quantity listed for the BomItem.| 0-n        |
| name                     | qualifiedNameType        | The reference designator (e.g., R1, C3) used in the Component element within Step.                                              | 1-1        |
| packageRef               | qualifiedNameType        | Name of the package defining physical outline and land-pattern for the component.                                               | 0-1        |
| populate                 | boolean                  | Indicates if the RefDes was populated ("true" or "false"). True is default.                                                   | 0-1        |
| layerRef                 | qualifiedNameType        | Reference to a layerFunction identifying where the component is mounted (top, bottom, internal).                                 | 0-1        |
| Tuning                   | TuningType               | Nested element specifying any tuning adjustments required for the BomItem.                                                        | 0-n        |
| Firmware                 | FirmwareType             | Nested element describing firmware or programmable data associated with the BomItem.                                             | 0-n        |

**Example:**
```xml
<RefDes name="U1" packageRef="SOIC12" populate="true" layerRef="TOP_COPPER">
    <Tuning value="ToleranceHigh" comments="Adjust pad width"/>
    <Firmware progName="Bootloader" progVersion="v1.1">
        <File name="boot.bin" crc="0x1A2B3C"/>
    </Firmware>
</RefDes>
```
