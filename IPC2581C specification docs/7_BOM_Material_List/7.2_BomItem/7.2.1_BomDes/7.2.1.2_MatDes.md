# 7.2.1.2 MatDes

The `MatDes` element is a BOM item designation used to identify materials contained in the BOM and referenced in the IPC-2581 file.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                       | Occurrence |
|--------------------------|--------------------------|-------------------------------------------------------------------------------------------------------------------|------------|
| MatDes                   | BomDesType               | A BOM item designation for materials contained in the BOM.                                                        | 0-n        |
| name                     | qualifiedNameType        | A unique name assigned as a designation for materials (combination of letters and numbers).                        | 1-1        |
| layerRef                 | qualifiedNameType        | Reference to a layer with a LayerFunction describing the materials being identified in the CadData.               | 0-1        |

**Example:**
```xml
<MatDes name="Mat1" layerRef="MATERIAL_LAYER"/>
```
