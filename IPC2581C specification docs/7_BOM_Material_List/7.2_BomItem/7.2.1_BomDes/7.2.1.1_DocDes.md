# 7.2.1.1 DocDes

The `DocDes` element is a BOM item designation used to identify documents contained in the BOM and referenced in the IPC-2581 file.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                                 | Occurrence |
|--------------------------|--------------------------|-----------------------------------------------------------------------------------------------------------------------------|------------|
| DocDes                   | BomDesType               | A BOM item designation for documents contained in the BOM.                                                                  | 0-n        |
| name                     | qualifiedNameType        | A unique name assigned as a designation for documents (combination of letters and numbers).                                 | 1-1        |
| layerRef                 | qualifiedNameType        | Reference to a layer with a DOCUMENT layerFunction containing graphical data considered documentation.                    | 0-1        |

**Example:**
```xml
<DocDes name="Doc1" layerRef="DOCUMENT_LAYER"/>
```
