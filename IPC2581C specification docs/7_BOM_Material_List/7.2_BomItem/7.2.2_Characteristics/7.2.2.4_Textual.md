# 7.2.2.4 Textual

The `Textual` element describes textual characteristics of a `BomItem`, capturing descriptive information and its source.

| Attribute / Element Name           | Attribute / Element Type | Description                                                                                                    | Occurrence |
|------------------------------------|--------------------------|----------------------------------------------------------------------------------------------------------------|------------|
| Textual                            | TextualType              | Nested element containing textual characteristics for a `BomItem`.                                              | 0-n        |
| definitionSource                   | string                   | Source of the textual characteristic information.                                                               | 0-1        |
| textualCharacteristicName         | string                   | Name of the textual characteristic (e.g., color, finish).                                                       | 0-1        |
| textualCharacteristicValue        | string                   | Textual value identified for the BOM item.                                                                      | 0-1        |

**Example:**
```xml
<Textual definitionSource="Engineering Notes" textualCharacteristicName="Finish" textualCharacteristicValue="Gold Plated"/>
```
