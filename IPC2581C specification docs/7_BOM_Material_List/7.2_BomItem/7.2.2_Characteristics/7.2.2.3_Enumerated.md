# 7.2.2.3 Enumerated

The `Enumerated` element describes enumerated value characteristics for a `BomItem`, including the source of the information.

| Attribute / Element Name              | Attribute / Element Type | Description                                                                                                         | Occurrence |
|---------------------------------------|--------------------------|---------------------------------------------------------------------------------------------------------------------|------------|
| Enumerated                            | EnumeratedType           | Nested element containing enumerated characteristics for a `BomItem`.                                                | 0-n        |
| definitionSource                      | string                   | Source of the enumerated characteristic information.                                                                 | 0-1        |
| enumeratedCharacteristicName         | string                   | Name of the enumerated characteristic.                                                                               | 0-1        |
| enumeratedCharacteristicValue        | string                   | Value of the enumerated characteristic.                                                                              | 0-1        |

**Example:**
```xml
<Enumerated definitionSource="Supplier Data" enumeratedCharacteristicName="RoHSCompliance" enumeratedCharacteristicValue="Yes"/>
```
