# 7.2.2.2 Ranged

The `Ranged` element describes tolerance or variation ranges for measurable characteristics of a `BomItem`, defining acceptable upper and lower limits.

| Attribute / Element Name           | Attribute / Element Type | Description                                                                                                              | Occurrence |
|------------------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------------|------------|
| Ranged                             | RangedType               | Nested element containing range characteristics for a `BomItem`.                                                         | 0-n        |
| definitionSource                   | string                   | Source of the range information.                                                                                         | 0-1        |
| rangedCharacteristicName          | string                   | Name of the ranged characteristic (e.g., thickness).                                                                     | 0-1        |
| rangedCharacteristicLowerValue    | double                   | Lower limit of the ranged characteristic.                                                                                | 0-1        |
| rangedCharacteristicUpperValue    | double                   | Upper limit of the ranged characteristic.                                                                                | 0-1        |
| engineeringUnitOfMeasure           | string                   | Engineering unit of measure for the ranged values.                                                                       | 0-1        |
| engineeringNegativeTolerance       | double                   | Negative tolerance applied to the range for target value.                                                                | 0-1        |
| engineeringPositiveTolerance       | double                   | Positive tolerance applied to the range for target value.                                                                | 0-1        |

**Example:**
```xml
<Ranged definitionSource="Design Spec" rangedCharacteristicName="Thickness" rangedCharacteristicLowerValue="1.5" rangedCharacteristicUpperValue="1.7" engineeringUnitOfMeasure="mm"/>
```
