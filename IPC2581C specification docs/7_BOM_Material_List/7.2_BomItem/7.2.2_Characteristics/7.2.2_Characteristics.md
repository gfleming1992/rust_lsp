# 7.2.2 Characteristics

A group of specific characteristics applicable to a particular `BomItem`; each characteristic relates to one category of the `BomItem`. Characteristics are defined under the major element `Characteristics`.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                                        | Occurrence |
|--------------------------|--------------------------|--------------------------------------------------------------------------------------------------------------------|------------|
| Characteristics          | CharacteristicsType      | A nested element containing descriptive strings linked together and references to external definitions.            | 1-1        |
| category                 | bomCategoryType          | Defines the category of the `BomItem` (ELECTRICAL, PROGRAMMABLE, MECHANICAL, MATERIAL, DOCUMENT).                  | 1-1        |
| Measured                 | MeasuredType             | Nested element describing measurable characteristics (nominal values and tolerances).                              | 0-n        |
| Ranged                   | RangedType               | Nested element describing allowable ranges and tolerances for characteristics.                                     | 0-n        |
| Enumerated               | EnumeratedType           | Nested element describing enumerated characteristic values and sources.                                            | 0-n        |
| Textual                  | TextualType              | Nested element describing textual characteristics and their source.                                                | 0-n        |

**Example:**
```xml
<Characteristics category="ELECTRICAL">
    <Measured definitionSource="Supplier Test" measuredCharacteristicName="Resistance" measuredCharacteristicValue="100" engineeringUnitOfMeasure="Ohms" engineeringNegativeTolerance="5" engineeringPositiveTolerance="5"/>
    <Ranged definitionSource="Design Spec" rangedCharacteristicName="Thickness" rangedCharacteristicLowerValue="1.5" rangedCharacteristicUpperValue="1.7" engineeringUnitOfMeasure="mm"/>
    <Enumerated definitionSource="Supplier Data" enumeratedCharacteristicName="RoHSCompliance" enumeratedCharacteristicValue="Yes"/>
    <Textual definitionSource="Engineering Notes" textualCharacteristicName="Finish" textualCharacteristicValue="Gold Plated"/>
</Characteristics>
```
