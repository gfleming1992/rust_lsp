# 5.4 Person

The `Person` element identifies an individual involved in the ordering and delivery process. It may be used to specify contact information and role associations for persons referenced elsewhere in the IPC-2581 file.

| Attribute / Element Name | Attribute / Element Type | Description                                                                                          | Occurrence |
|--------------------------|--------------------------|------------------------------------------------------------------------------------------------------|------------|
| Person                   | PersonType               | Identifies the person involved in sending, receiving, or having any role in the trading partner relationship. | 1-n        |
| name                     | string                   | The full name of the person.                                                                         | 1-1        |
| title                    | string                   | The job title, position, or role of the person within the enterprise.                                  | 0-1        |
| phone                    | string                   | The direct phone number for the person.                                                              | 0-1        |
| fax                      | string                   | The fax number for the person.                                                                       | 0-1        |
| email                    | string                   | The email address for the person.                                                                    | 0-1        |
| roleRef                  | IDREF                    | Reference to a defined `Role` element by its `name` attribute to indicate the person's function.     | 0-1        |

**Example:**
```xml
<LogisticHeader>
  <Role name="BUYER" description="Responsible for ordering parts"/>
  <Enterprise id="Acme" name="Acme Tool and Die Company Inc." code="1433" codeType="DUNS" address1="7347 Concorde Ave." city="Camden" stateProvince="NJ" country="US" postalCode="08780" phone="609-458-5943" email="AcmeCorp@mindspring.com"/>
  <Person name="John Doe" title="Procurement Manager" phone="+1-609-123-4567" email="jdoe@acme.com" roleRef="BUYER"/>
  <Person name="Jane Smith" title="Production Supervisor" phone="+1-609-987-6543" fax="+1-609-987-6544" email="jsmith@acme.com"/>
</LogisticHeader>
```
