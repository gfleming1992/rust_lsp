# 3.5.6.2 FillDescRef

The FillDescRef element is used throughout the IPC-2581 file to establish the relationship to a previously defined FillDesc. The Units of the predefined FillDesc must match the Units of the Ecad section in which it is instantiated.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| FillDescRef             | FillDescRefType         | An element used to reference a previously defined FillDesc, contained in the DictionaryFillDesc. | 1-n |
| id                      | qualifiedNameType       | The identification of the FillDesc being referenced from the DictionaryFillDesc. | 1-1 |
