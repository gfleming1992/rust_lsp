# 3.5.10 StandardShape

The StandardShape substitution group permits the substitution of any StandardPrimitive shape according to its individual description. A predefined StandardPrimitive may be instantiated by its unique "id" when contained in DictionaryStandard. When referencing a predefined primitive, the units must match.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| StandardPrimitive       | ABSTRACT                | Permits substitution of StandardPrimitive element with any of the sixteen standard primitive types. | 0-n |
| StandardPrimitiveRef    | StandardPrimitiveRefType| Reference to a predefined StandardPrimitive in DictionaryStandard by unique "id". Units must match. | 0-n |
