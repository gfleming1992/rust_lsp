# 3.5.3.1 CachedFirmware

The CachedFirmware element is used to describe firmware that will be contained in the DictionaryFirmware.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| CachedFirmware          | CachedFirmwareType      | The firmware description needed by a particular component that becomes part of the predefined firmware in the DictionaryFirmware. | 1-1 |
| hexEncodedBinary        | string                  | An attribute defining the binary code that shall be added to a particular component and which is contained in the DictionaryFirmware. | 1-1 |
