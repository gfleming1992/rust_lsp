# 3.5.12 UserShape

The UserShape substitution group permits the substitution of any UserPrimitive shape according to its individual description. A predefined UserPrimitive may be instantiated by its unique "id" when contained in DictionaryUser. When referencing a predefined primitive, the units must match.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| UserPrimitive           | ABSTRACT                | Permits substitution of UserPrimitive element with any user primitive description or type. | 0-n |
| UserPrimitiveRef        | UserPrimitiveRefType    | Reference to a predefined UserPrimitive in DictionaryUser by unique "id". Units must match. | 0-n |
