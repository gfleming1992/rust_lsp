# 3.5.11.2 UserSpecial

UserSpecial may be any combination of StandardShapes or UserShapes, used to develop logos, targets, drawing formats, or other combinations. Feature is a substitution group consisting of any graphic allowed by StandardShape or UserShape substitution groups.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| UserSpecial             | UserSpecialType         | Combination of primitive shapes organized as needed for special graphics. | 0-n |
| Feature                 | ABSTRACT                | Substitution group for any graphic allowed by StandardShape or UserShape. | 0-n |
