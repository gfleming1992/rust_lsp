# 3.5.11 UserPrimitive

The UserPrimitive substitution group consists of any simple graphic feature (Arc, Line, Outline, Polyline), as well as text or UserSpecial shapes. UserSpecial is a collection of Features, permitting logos, special targets, drawing formats, or other graphics. UserPrimitives can be predefined, assigned a unique "id", and contained in DictionaryUser. DictionaryUser defines the units used for graphic shapes.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Simple                  | ABSTRACT                | Defines an Arc, Line, Outline, or Polyline. Used in DfxMeasurement, Glyph, Slot, and other graphics. | 0-n |
| Text                    | TextType                | Defines text strings, fonts, bounding box, and Xform for position, mirror, or rotation. | 0-n |
| UserSpecial             | UserSpecialType         | Collection of Features, may develop any combination of graphical shapes. | 0-n |
