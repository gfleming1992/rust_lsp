# 3.5.9.9 Octagon

An Octagon is an eight-sided StandardPrimitive shape with equal sides and angles. Oriented with a point facing North. Defined by the length across the points.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Octagon                 | OctagonType             | Eight equal sides, defined by length across points. | 1-1 |
| length                  | nonNegativeDoubleType   | Length between two opposing corner points. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Octagon length="12.8">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Octagon>
```
