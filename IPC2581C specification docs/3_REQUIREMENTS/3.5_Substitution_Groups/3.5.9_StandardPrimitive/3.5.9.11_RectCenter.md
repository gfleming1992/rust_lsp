# 3.5.9.11 RectCenter

A RectCenter is a primitive shape that defines a rectangle by width and height, centered at the origin. Used to represent a square if width equals height.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| RectCenter              | RectCenterType          | Rectangle defined by width and height, centered at origin. | 1-1 |
| width                   | nonNegativeDoubleType   | Length of the rectangle about the x-axis. | 1-1 |
| height                  | nonNegativeDoubleType   | Length of the rectangle about the y-axis. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<RectCenter width="16.8" height="6.4">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectCenter>
```
