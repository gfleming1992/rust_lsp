# 3.5.9.1 Butterfly

A Butterfly is a StandardPrimitive shape with either a ROUND or SQUARE periphery, with two quadrants removed (0–90° and 180–270°). The round shape is defined by its diameter; the square by equal side dimension. The point of origin is the center of the Butterfly.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Butterfly               | ButterflyType           | Primitive shape (ROUND or SQUARE) with two quadrants removed. | 1-1 |
| shape                   | butterflyShapeType      | ROUND: like circleDef; SQUARE: like RectCenterDef. | 1-1 |
| diameter                | nonNegativeDoubleType   | Diameter for circular butterfly. | 0-1 |
| side                    | nonNegativeDoubleType   | Width and height for square butterfly. | 0-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Butterfly shape="ROUND" diameter="3.2">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Butterfly>
<Butterfly shape="SQUARE" side="1.8">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Butterfly>
```
