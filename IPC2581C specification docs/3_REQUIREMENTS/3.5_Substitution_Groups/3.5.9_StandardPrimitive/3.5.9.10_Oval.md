# 3.5.9.10 Oval

An Oval is a StandardPrimitive shape that defines a rectangle with a complete radius (180Â° arc) at each end. The center of the rectangle is the point of origin.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Oval                    | OvalType                | Rectangle with a complete radius at each end. | 1-1 |
| width                   | nonNegativeDoubleType   | Length of the rectangle about the x-axis. | 1-1 |
| height                  | nonNegativeDoubleType   | Length of the rectangle about the y-axis. Must not exceed width. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Oval width="10.6" height="6.4">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Oval>
```
