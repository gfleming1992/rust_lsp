# 3.5.9.14 RectRound

A RectRound is a StandardPrimitive shape that defines a rectangle with rounded corners. The rectangle is defined by width and height, centered at the origin. Used to represent a square with rounded corners if width equals height.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| RectRound               | RectRoundType           | Rectangle with one or more rounded corners. | 1-1 |
| width                   | nonNegativeDoubleType   | Length of the rectangle about the x-axis. | 1-1 |
| height                  | nonNegativeDoubleType   | Length of the rectangle about the y-axis. | 1-1 |
| radius                  | nonNegativeDoubleType   | Radius for corners. Must not exceed ½ height or ½ width. | 1-1 |
| upperRight              | boolean                 | Upper right corner rounded. | 0-1 |
| upperLeft               | boolean                 | Upper left corner rounded. | 0-1 |
| lowerLeft               | boolean                 | Lower left corner rounded. | 0-1 |
| lowerRight              | boolean                 | Lower right corner rounded. | 0-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<RectRound width="10.2" height="6.4" radius="3.2" upperRight="true" upperLeft="false" lowerLeft="true" lowerRight="false">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectRound>
<RectRound width="4.8" height="4.8" radius="2.4" upperRight="true" upperLeft="false" lowerLeft="false" lowerRight="true">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectRound>
```
