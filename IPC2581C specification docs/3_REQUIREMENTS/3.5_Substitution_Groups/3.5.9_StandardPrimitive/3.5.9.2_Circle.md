# 3.5.9.2 Circle

A Circle is a StandardPrimitive shape defined by its diameter. The point of origin is the center of the circle.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Circle                  | CircleType              | Defines a circular shape by diameter. | 1-1 |
| diameter                | nonNegativeDoubleType   | Diameter of the circle. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Circle diameter="3.6">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Circle>
<Circle diameter="3.6">
  <LineDesc lineEnd="ROUND" lineWidth="0.1"/>
  <FillDesc fillProperty="HATCH" lineWidth="0.02" pitch1="0.08" angle1="135">
    <Color r="0" g="0" b="200"/>
  </FillDesc>
</Circle>
```
