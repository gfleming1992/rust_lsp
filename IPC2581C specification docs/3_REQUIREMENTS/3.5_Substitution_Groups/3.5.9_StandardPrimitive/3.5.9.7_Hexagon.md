# 3.5.9.7 Hexagon

A Hexagon is a six-sided StandardPrimitive shape with equal sides and angles. Oriented with a point facing North. Defined by the length across the points.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Hexagon                 | HexagonType             | Six equal sides, defined by length across points. | 1-1 |
| length                  | nonNegativeDoubleType   | Length between two opposing corner points. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Hexagon length="12.8">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Hexagon>
```
