# 3.5.9.16 Triangle

A Triangle is a StandardPrimitive shape that is an isosceles triangle with two equal sides and a base. Defined by base and height dimensions. The point of origin is at the center of the base and height.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Triangle                | TriangleType            | Isosceles triangle defined by base and height. | 1-1 |
| base                    | nonNegativeDoubleType   | Distance between two corner points of the base. | 1-1 |
| height                  | nonNegativeDoubleType   | Height of the triangle. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Triangle base="4.0" height="8.0">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Triangle>
```
