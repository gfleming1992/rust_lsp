# 3.5.9.13 RectCorner

A RectCorner is a StandardPrimitive shape that defines a rectangle by its lower left and upper right corners. The point of origin is (0,0).

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| RectCorner              | RectCornerType          | Rectangle defined by lower left and upper right corners. | 1-1 |
| lowerLeftX              | double                  | Lower left x dimension. | 1-1 |
| lowerLeftY              | double                  | Lower left y dimension. | 1-1 |
| upperRightX             | double                  | Upper right x dimension. | 1-1 |
| upperRightY             | double                  | Upper right y dimension. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<RectCorner lowerLeftX="0.0" lowerLeftY="0.0" upperRightX="12.6" upperRightY="6.8">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectCorner>
```
