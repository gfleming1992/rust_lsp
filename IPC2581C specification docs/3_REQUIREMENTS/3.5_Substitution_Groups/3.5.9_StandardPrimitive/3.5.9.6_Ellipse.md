# 3.5.9.6 Ellipse

An Ellipse is a StandardPrimitive shape defined by width and height. The point of origin is the center of the width and height dimensions.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Ellipse                 | EllipseType             | Elliptical shape defined by width and height. | 1-1 |
| width                   | nonNegativeDoubleType   | Height of the ellipse on the y-axis. | 1-1 |
| height                  | nonNegativeDoubleType   | Width of the ellipse on the x-axis. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Ellipse width="12.60" height="6.20">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Ellipse>
```
