# 3.5.9.3 Contour

A Contour is a StandardPrimitive shape that defines a sequence of connected edges forming a polygon. Edges can be straight or circular. The polygon is a closed shape whose edges do not cross. Cutout represents the absence of material inside the polygon shape.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Contour                 | ContourType             | Sequence of connected edges forming a polygon. | 1-1 |
| Polygon                 | PolygonType             | Closed shape, coordinates defined relative to local coordinate system. | 1-1 |
| Cutout                  | CutoutType              | Closed shape representing absence of material within the polygon. | 0-n |

**Example:**
```xml
<Contour>
  <Polygon>
    <PolyBegin x="0.00" y="0.00"/>
    <PolyStepSegment x="0.00" y="7.00"/>
    <PolyStepSegment x="-8.00" y="7.00"/>
    <PolyStepCurve x="-15.00" y="0.00" centerX="-8.00" centerY="0.00" clockwise="false"/>
    <PolyStepSegment x="0.00" y="0.00"/>
    <FillDesc fillProperty="FILL">
      <Color r="150" g="0" b="0"/>
    </FillDesc>
  </Polygon>
</Contour>

<Contour>
  <Polygon>
    <PolyBegin x="0.00" y="0.00"/>
    <PolyStepSegment x="-14.00" y="0.00"/>
    <PolyStepCurve x="-3.00" y="3.00" centerX="-14.00" centerY="3.00" clockwise="true"/>
    <PolyStepSegment x="-17.00" y="7.00"/>
    <PolyStepSegment x="0.00" y="7.00"/>
    <PolyStepSegment x="0.00" y="0.00"/>
    <FillDesc fillProperty="FILL">
      <Color r="150" g="0" b="0"/>
    </FillDesc>
  </Polygon>
  <Cutout>
    <PolyBegin x="-10.00" y="5.00"/>
    <PolyStepSegment x="-13.00" y="5.00"/>
    <PolyStepSegment x="-13.00" y="3.00"/>
    <PolyStepCurve x="-10.00" y="3.00" centerX="-11.50" centerY="3.00" clockwise="false"/>
    <PolyStepSegment x="-10.00" y="5.00"/>
    <FillDesc fillProperty="VOID"/>
  </Cutout>
</Contour>
```
