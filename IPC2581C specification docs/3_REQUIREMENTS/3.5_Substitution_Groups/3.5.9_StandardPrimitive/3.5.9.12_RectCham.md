# 3.5.9.12 RectCham

A RectCham is a StandardPrimitive shape that defines a rectangle with chamfered corners. The rectangle is defined by width and height, centered at the origin. Used to represent a square with chamfered corners if width equals height.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| RectCham                | RectChamType            | Rectangle with one or more chamfered corners. | 1-1 |
| width                   | nonNegativeDoubleType   | Length of the rectangle about the x-axis. | 1-1 |
| height                  | nonNegativeDoubleType   | Length of the rectangle about the y-axis. | 1-1 |
| chamfer                 | nonNegativeDoubleType   | Length from each corner defining chamfer. Must not exceed ½ height or ½ width. | 1-1 |
| upperRight              | Boolean                 | Upper right corner chamfered. | 0-1 |
| upperLeft               | Boolean                 | Upper left corner chamfered. | 0-1 |
| lowerLeft               | Boolean                 | Lower left corner chamfered. | 0-1 |
| lowerRight              | Boolean                 | Lower right corner chamfered. | 0-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<RectCham width="10.6" height="6.2" chamfer="2.0" upperRight="false" upperLeft="true" lowerLeft="false" lowerRight="true">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectCham>
<RectCham width="8.4" height="8.4" chamfer="2.0" upperRight="false" upperLeft="true" lowerLeft="false" lowerRight="true">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</RectCham>
```
