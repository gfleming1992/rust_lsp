# 3.5.9.4 Diamond

A Diamond is a four-sided StandardPrimitive shape with equal sides, specified by width and height. The diamond is positioned with one corner facing North.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| Diamond                 | DiamondType             | Four equal sides, converging to points. Controlled by width and height. | 1-1 |
| width                   | nonNegativeDoubleType   | Length along the x-axis, centered. | 1-1 |
| height                  | nonNegativeDoubleType   | Length along the y-axis, centered. | 1-1 |
| LineDescGroup           | LineDescGroupType       | Substitution group for LineDesc or LineDescRef. | 0-1 |
| FillDescGroup           | FillDescGroupType       | Substitution group for FillDesc or FillDescRef. | 0-1 |

**Example:**
```xml
<Diamond width="10.40" height="6.20">
  <FillDesc fillProperty="FILL">
    <Color r="150" g="0" b="0"/>
  </FillDesc>
</Diamond>
```
