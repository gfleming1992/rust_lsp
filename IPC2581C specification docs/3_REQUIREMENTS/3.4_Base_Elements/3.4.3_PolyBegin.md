# 3.4.3 PolyBegin

The PolyBegin element defines the starting point of Polygon or Polyline element.

| Attribute / Element Name | Attribute / Element Type | Description | Occurrence |
|-------------------------|-------------------------|-------------|------------|
| PolyBegin               | PolyBeginType           | The PolyBegin element defines the starting point of a polygon or polyline. | 1-1 |
| x                       | Double                  | The X co-ordinate starting point. | 1-1 |
| y                       | Double                  | The Y co-ordinate starting point. | 1-1 |
